## Nasıl Kullanılır?
# JS
```javascript
const db = require("foe.db");

// Db Veri Çekme - Kaydetme
db.set("veri.1", true);
db.get("veri.1");
db.fetch("veri.1");

// Veri Silme

db.delete("key");
db.deleteAll();

```
